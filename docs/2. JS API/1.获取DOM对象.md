

### ❗根据CSS选择器来获取DOM元素

```js
document.querySelector('CSS选择器')	// 只获取第一个匹配的
```

```js
document.querySelector('CSS选择器')	// 获取全部匹配的
```



### 其他获取DOM元素的方法

```js
document.getElementById("nav")	// 根据 id 获取一个元素
```

```js
document.getElementsByTagName("div")	// 根据标签获取一类元素 获取页面所有 div
```

```js
document.getElementsByClassName("w")	// 根据类名获取元素 获取页面所有类名为 w 的元素
```
