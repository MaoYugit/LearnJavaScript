## BOM 概述

**BOM（Browser Object Model）** 是**浏览器对象模型**，它提供了独立于网页内容而与浏览器窗口进行交互的对象。

我的核心理解是：
- **DOM** 关注的是**文档**（HTML/XML内容）
- **BOM** 关注的是**浏览器**本身和浏览器窗口

BOM没有官方标准（不同于DOM的W3C标准），但所有现代浏览器都实现了相似的功能和对象。

## BOM 的核心对象

### 1. window 对象
**window** 是BOM的顶层对象，也是JavaScript在浏览器中的全局对象。

```javascript
// window 的属性和方法
console.log(window.innerWidth);  // 视口宽度
console.log(window.innerHeight); // 视口高度

// 窗口操作
window.open('https://example.com', '_blank'); // 打开新窗口
window.close(); // 关闭当前窗口（有限制）
window.resizeTo(800, 600); // 调整窗口大小

// 对话框
window.alert('这是一个警告');
const result = window.confirm('您确定吗？');
const input = window.prompt('请输入内容：');
```

### 2. navigator 对象
提供有关**浏览器和操作系统**的信息。

```javascript
console.log(navigator.userAgent);     // 浏览器用户代理字符串
console.log(navigator.platform);      // 操作系统平台
console.log(navigator.language);      // 浏览器语言
console.log(navigator.cookieEnabled); // 是否启用cookie
console.log(navigator.onLine);        // 是否在线

// 现代API
if (navigator.geolocation) {
    navigator.geolocation.getCurrentPosition(position => {
        console.log('纬度:', position.coords.latitude);
        console.log('经度:', position.coords.longitude);
    });
}

if (navigator.mediaDevices) {
    // 访问摄像头、麦克风等
}
```

### 3. location 对象
包含当前URL的信息，并允许**导航和重定向**。

```javascript
// 获取URL信息
console.log(location.href);      // 完整URL
console.log(location.protocol);  // 协议 (https:)
console.log(location.host);      // 主机名和端口
console.log(location.hostname);  // 主机名
console.log(location.pathname);  // 路径
console.log(location.search);    // 查询参数
console.log(location.hash);      // 锚点

// 页面导航
location.href = 'https://new-site.com'; // 跳转到新页面
location.reload();                      // 重新加载
location.replace('https://new-site.com'); // 替换当前历史记录
```

### 4. history 对象
管理**浏览历史记录**。

```javascript
// 历史记录操作
history.back();     // 后退一页
history.forward();  // 前进一页
history.go(-2);     // 后退两页
history.go(1);      // 前进一页

// 现代单页应用常用
history.pushState({page: 1}, 'Title 1', '/page1'); // 添加历史记录，不刷新页面
history.replaceState({page: 2}, 'Title 2', '/page2'); // 替换当前历史记录
```

### 5. screen 对象
提供**用户屏幕**的信息。

```javascript
console.log(screen.width);        // 屏幕宽度
console.log(screen.height);       // 屏幕高度
console.log(screen.availWidth);   // 可用宽度（减去任务栏等）
console.log(screen.availHeight);  // 可用高度
console.log(screen.colorDepth);   // 颜色深度
console.log(screen.pixelDepth);   // 像素深度
```

## BOM 的实际应用场景

### 1. 响应式设计
```javascript
// 监听窗口大小变化
window.addEventListener('resize', () => {
    console.log(`窗口大小: ${window.innerWidth} x ${window.innerHeight}`);
    
    if (window.innerWidth < 768) {
        // 移动端逻辑
    } else {
        // 桌面端逻辑
    }
});
```

### 2. 单页应用路由
```javascript
// 监听URL变化
window.addEventListener('popstate', (event) => {
    // 处理浏览器前进/后退
    handleRouteChange(location.pathname);
});

// 编程式导航
function navigateTo(path) {
    history.pushState({}, '', path);
    handleRouteChange(path);
}
```

### 3. 性能监控
```javascript
// 页面加载性能
window.addEventListener('load', () => {
    const timing = performance.timing;
    const loadTime = timing.loadEventEnd - timing.navigationStart;
    console.log(`页面加载耗时: ${loadTime}ms`);
});
```

### 4. 设备能力检测
```javascript
// 检测设备能力
const capabilities = {
    touch: 'ontouchstart' in window,
    geolocation: 'geolocation' in navigator,
    serviceWorker: 'serviceWorker' in navigator,
    webGL: !!window.WebGLRenderingContext
};
```

## 重要特性

### 1. 定时器
```javascript
// 一次性定时器
const timeoutId = setTimeout(() => {
    console.log('3秒后执行');
}, 3000);

// 周期性定时器
const intervalId = setInterval(() => {
    console.log('每1秒执行一次');
}, 1000);

// 清除定时器
clearTimeout(timeoutId);
clearInterval(intervalId);
```

### 2. 本地存储
```javascript
// localStorage - 持久化存储
localStorage.setItem('user', JSON.stringify({name: 'John'}));
const user = JSON.parse(localStorage.getItem('user'));

// sessionStorage - 会话级存储
sessionStorage.setItem('token', 'abc123');
```

## 总结

我对BOM的理解是：

1. **定位**：BOM是浏览器功能的编程接口，关注浏览器本身而非文档内容
2. **核心**：以window对象为根，包含navigator、location、history、screen等关键对象
3. **用途**：实现页面导航、历史管理、设备检测、窗口控制等浏览器级功能
4. **重要性**：在现代Web开发中，BOM是实现单页应用、响应式设计、性能优化的基础

虽然BOM没有官方标准，但各浏览器厂商都遵循了事实上的标准，使其成为开发现代Web应用不可或缺的一部分。
